Doesn't look like this is installed on this computer yet. Here's exactly what will happen, before you start — nothing here should surprise you:

1. You'll download one file.

It's a setup script, not an app store install. Your browser may warn you that it's an unrecognized file — that's normal for any script you download directly rather than through an app store; you're choosing to run it.

2. You'll run it, and a black window will briefly appear.

That's the setup program working — it copies a few files to your computer and checks whether you already have a hardware-monitoring tool called LibreHardwareMonitor. If you do, it reuses it. If not, it downloads it from its official, open-source source.

3. Windows may ask for your permission (a security prompt).

This can happen when the hardware-monitoring tool needs deeper access to read things like CPU temperature. If you see this, it's expected — click Yes to allow it. If you'd rather not, you can decline; tracking still works, just without temperature/fan/voltage readings.

4. A shortcut named "Amit" appears on your desktop.

That opens your daily Amit Hub — sign-in, pursuits, and everything else, not just this tracker. Once you're signed in there, come to Computer Health and use the Launch Tracker button any time.

5. Nothing runs in the background unless you start it.

Closing this tab, or clicking the Tracker button at the top of this page, stops everything again.
