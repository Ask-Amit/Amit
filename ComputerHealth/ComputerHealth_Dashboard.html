Doesn't look like this is installed on this computer yet

Two small programs get installed, nothing else, and here's exactly what each one is for:

🖥️ The Amit Tracker — talks to this dashboard so it can show you live data about this computer. Adds one icon to your desktop.
🌡️ LibreHardwareMonitor — a well-known, open-source tool that reads your computer's temperature, fan speed, and voltage sensors. Amit uses it as the connection to that hardware data; it's not something Amit built, just something Amit relies on. If you already have it, nothing new downloads — yours gets reused.

Here's exactly what will happen, step by step, before you start — nothing here should surprise you:

1. You'll download one file: Amit Installer.

Not from an app store, so your browser will very likely warn you it's an unrecognized file — this happens in Edge, Chrome, and every other major browser for any brand-new program, not just this one. Here's the exact click sequence, since the real "trust it" option hides in an easy-to-miss spot:

In the small download popup: click the "…" (three dots) next to the file → choose Keep.

A second, bigger warning box then appears ("Make sure you trust..."). The real action is not the big "Delete" button — click the small ▾ arrow next to Delete, and choose Keep anyway (Chrome) or the equivalent option there. This two-step confirmation is genuinely confusing and not something we can skip — it only goes away once this installer has a paid security certificate behind it, which isn't set up yet.

2. Double-click it to run — a black window will briefly appear.

That's the installer working — it copies a few files to your computer and checks whether you already have a hardware-monitoring tool called LibreHardwareMonitor. If you do, it reuses it. If not, it downloads it from its official, open-source source. A message box confirms when it's done.

3. Windows may ask for your permission (a security prompt).

This can happen when the hardware-monitoring tool needs deeper access to read things like CPU temperature. If you see this, it's expected — click Yes to allow it. If you'd rather not, you can decline; tracking still works, just without temperature/fan/voltage readings.

4. A shortcut named "Amit" appears on your desktop.

That opens your daily Amit Hub — sign-in, pursuits, and everything else, not just this tracker. Once you're signed in there, come to Computer Health and use the Launch Tracker button any time.

5. Nothing runs in the background unless you start it.

Closing this tab, or clicking the Tracker button at the top of this page, stops everything again.
